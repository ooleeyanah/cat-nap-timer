# Cat Nap Timer v1.0.0

A cute and functional timer app for your power naps! 

## Features
- Multiple timer presets
- Adorable cat meowing sound when time's up
- Clean, simple interface
- Custom timer duration support

## Download
- **macOS**: Download the `.dmg` file for easy installation
- **All platforms**: Download the `.zip` file and extract

## Installation
- **macOS DMG**: Double-click to mount, drag to Applications folder
- **ZIP**: Extract and run the executable
